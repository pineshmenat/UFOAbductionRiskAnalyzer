# UFOAbductionRiskAnalyzer
UFO Abduction Risk Analyzer is a Web application for finding one's risk of abduction based on real time user location. Website plots all the UFOs from National UFO Reporting Center (NUFORC) on Google maps. Each UFO object provides necessary details like shape, location and weather details.
